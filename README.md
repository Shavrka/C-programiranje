# C-programiranje
Zadaci urađeni u C programskom jeziku. Uz programe će blagovremeno biti opisi programa, kako rade i šta rade. Biće tekstovi zadataka i njihova riješenja, kao i opis koda tamo dje bude potrebno. Svrha ovog je moje vježbanje i usavršavanje programiranja u C jeziku, a i možda pomogne nekome.
